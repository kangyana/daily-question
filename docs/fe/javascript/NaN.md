### 6种基本数据类型。

- String 字符串
- Number 数字
- Boolean 布尔值
- Null 空
- Undefined 未定义
- Symbol （es6新增）

### 5种引用数据

- Object 对象
- Array 数组
- Function 函数
- RegExp 正则
- Date 日期
